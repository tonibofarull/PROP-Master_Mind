# PROP-Master_Mind

Implementation of MasterMind (board game).

### Prerequisites

Java 8

## Compile & Execute

To compile:

```
javac -encoding UTF8 Main.java
```

To execute:

```
java Main
```

Or use the '.jar' that can be found in 'exe'.
